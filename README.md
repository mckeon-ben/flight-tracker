# flight-tracker :airplane::heart:
A Jupyter notebook which generates a timelapse of flights taken by [@ltilke](https://github.com/ltilke) and I as a diary of our long distance love story.

## Requirements
- [pandas](https://github.com/pandas-dev/pandas)
- [matplotlib](https://github.com/matplotlib/matplotlib)
- [cartopy](https://github.com/SciTools/cartopy)
- [airportsdata](https://github.com/mborsetti/airportsdata)
